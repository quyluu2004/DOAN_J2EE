# Danh mục Công nghệ

## Backend
- **Cốt lõi:** Java 17+, Spring Boot 3.x
- **ORM:** Hibernate / Spring Data JPA
- **Cơ sở dữ liệu:** MySQL (lưu trữ trên Aiven)
- **Bộ nhớ đệm:** Redis (lưu trữ trên Aiven/Render)
- **Bảo mật:** Spring Security, JWT, Discord 2FA
- **Ghi nhật ký:** SLF4J / Logback
- **Công cụ xây dựng:** Maven

## Frontend
- **Cốt lõi:** React 18+
- **Công cụ xây dựng:** Vite
- **Định dạng:** Vanilla CSS / Tailwind (một số tham chiếu)
- **Hiệu ứng:** Framer Motion
- **Biểu tượng:** Lucide React
- **Quản lý trạng thái:** React Context API (Auth, Cart, Localization)
- **Thư viện HTTP:** Axios

## Hạ tầng
- **Hosting:** Render (Gói miễn phí)
- **Tài nguyên:** Cloudinary (Ảnh, Mô hình 3D)
- **Hosting DB:** Aiven
